# 🎓 StudentSphere

A modern Student Management System built with **Flask**, **Python**, **SQLAlchemy**, **Bootstrap 5**, and **SQLite**.

## ✨ Features

- 🔐 Admin Login & Signup
- 🔑 Forgot Password with Email OTP
- 👨‍🎓 Add Student
- ✏️ Edit Student
- 🗑️ Delete Student
- 👤 Student Profile
- 🔍 Search Students
- 🎯 Filter by Course, Semester & Gender
- 📄 Export Reports (Excel & PDF)
- 📥 Import Students (Excel & CSV)
- 🌙 Dark Mode
- 📊 Dashboard Analytics
- 👤 Admin Profile Management

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/StudentSphere.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python app.py
```
